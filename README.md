## Getting Started
### Installation
1. Clone the repo
    ```
    git clone https://github.com/DhayananthK/fabheads-assignment.git
    ```
2. Installing Pip Packages
    ```
    pip install -r requirements.txt
    ```
3. Run the Application
    ```
    python run.py
    ```
